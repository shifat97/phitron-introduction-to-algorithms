# Introduction to Algorithms - Phitron

This repository contains my solutions and implementations for the **Introduction to Algorithms** course by **Phitron**. It covers a wide range of fundamental algorithms, with a heavy focus on graph theory, dynamic programming, and problem-solving.

## 🚀 Repository Description
A comprehensive collection of algorithm implementations, practice problems, and assignments from the Phitron "Introduction to Algorithms" course, primarily in C++ with supplementary Python and Java examples.

---

## 📂 Project Structure

The repository is organized into modules, each focusing on a specific algorithmic concept:

### 1. Graph Basics & Representations
- **01_Introduction_to_Algorithms**: Basic graph types and input representations (Adjacency Matrix, Adjacency List, Edge List).
- **02_Breadth_First_Search_BFS**: Implementation of BFS, vertex sum, shortest distance, and path tracking.
- **03_Depth_First_Search_DFS**: Implementation of DFS, and applying both BFS and DFS on 2D grids.

### 2. Advanced Graph Algorithms
- **05_Graph_Cycle**: Cycle detection in both directed and undirected graphs using BFS and DFS.
- **06_Dijkstra_Algorithm**: Shortest path algorithm for weighted graphs (Naive and Optimized versions).
- **08_Bellman_Ford_Algorithm**: Handling negative edge weights and negative cycle detection.
- **09_Floyd_Warshall_Algorithm**: All-pairs shortest path algorithm.
- **10_Disjoint_Set_Union (DSU)**: Path compression and union by rank/size for cycle detection and connectivity.

### 3. Dynamic Programming (DP)
- **12_Dynamic_Programming**: Introduction to DP, including Fibonacci (Top-down and Bottom-up).
- **14_Knapsack**: Classic 0/1 Knapsack problem and its DP optimizations.

### 4. Assignments & Exams
This repository also contains solutions for course-specific evaluations:
- **04_Assignment**: Connected components and grid traversal problems.
- **07_Midterm_Exam**: Area of components, Knight's move, and maze navigation.
- **11_Assignment**: Shortest distance and cycle detection practice.
- **15_Final_Assignment**: Comprehensive problem set covering all course topics.
